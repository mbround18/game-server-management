§Game Server Backup Utility

This crate provides functionality for creating compressed tarball backups of game server data. It is designed to be a reusable component within the Game Server Management (GSM) workspace.

The primary function, backup, takes an input directory and an output path, and creates a .tar.gz archive of the directory’s contents. It includes features for skipping certain files, such as auto-backups, to avoid redundant data in the archives.
