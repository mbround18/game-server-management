§Proton Management

This module provides functionality for finding, downloading, and configuring Proton, Valve’s compatibility tool for running Windows games on Linux. It is a key component for enabling Windows-based game servers to run in a Linux environment.

The module can automatically locate installed Proton versions, download specific versions from GitHub, and set up the necessary environment for a game server to use Proton.
