Expand description

Spawns a job to run on a cron-like schedule asynchronously.

This function takes a cron schedule string and a closure, and spawns a tokio task to execute the closure at the specified times. The schedule is based on UTC.

§Arguments

  • schedule_str: A string representing the cron schedule (e.g., “0 0 * * * *”).
  • job: A closure that will be executed when the schedule is met. The closure must be Send, Sync, and have a 'static lifetime.

§Panics

This function does not panic, but it will log an error if the schedule string is invalid.

§Example

use gsm_cron::spawn_scheduled_job;

// Schedule a job to run every minute.
spawn_scheduled_job("0 * * * * *".to_string(), || {
    println!("This job runs every minute!");
});
